The Personal Info Front Page

Feature: Add Personal Info


this feature adds new personal information in the below stackpanel
All fields are to be Not Null (excl Gender)

Feature: Edit Personal Info


this feature Edits personal information from the below stackpanel
a Record Needs to selected for this to work
all fields need to be filled in (excl Gender)

KNote- update won't work unless all fields are completed, then the record to be updated is selected

Feature: Delete PersonalInfo


this feature Deletes personal information from the below stackpanel
a Record Needs to selected for this to work

Note: - update won't work if all fields are not filled out
