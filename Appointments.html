Feature: Add Appointments


this feature adds new appointments in the below stackpanel

all other fields are to be Not Null

Feature: Update Appointments


this feature Updates appointments from the below stackpanel
a Record Needs to selected for this to work
all fields need to be filled in
Description: needs to be Unique
all other fields are to be Not Null

Feature: Delete Appointments


this feature Deletes appointments from the below stackpanel
a Record Needs to selected for this to work

Known bug - update won't work if all fields are not filled out
